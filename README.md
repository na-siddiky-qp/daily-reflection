## 🚀 Setup & Run

```bash
# clone repo
git clone https://github.com/na-siddiky-qp/daily-reflection.git
cd daily-reflection

# install dependencies
npm install

# run locally
npm run dev

# build for production
npm run build

# preview production build
npm run preview
```
